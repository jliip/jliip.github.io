# jliip.github.io

Minimal personal website powered by VitePress.

## Development

- Install dependencies: `npm install`
- Start dev server: `npm run dev`
- Build for production: `npm run build`
- Preview build: `npm run preview`

## Content

All site content lives in the `docs/` folder.

- `docs/index.md` is the landing page.
- `docs/about.md`, `docs/projects.md`, `docs/blog.md`, `docs/research.md` are section pages.
- Sidebar links are configured in `docs/.vitepress/config.ts`.

Keep content short and replace placeholders as needed.
